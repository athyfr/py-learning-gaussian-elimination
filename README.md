# Gaussian Elimination Learning Project

This is a project I'm working on for school, as part of the same course as
[Python Calculator Project.](https://github.com/athyfr/python-tk-calculator-project)

This repository contains:

- `matrix.py`: A Matrix class implementing:
  - Augmented and non-augmented matrices.
  - Elementary row operations, and their inverses (member functions).
  - Gaussian elimination (member function).
- `main.py`: A TUI for testing `matrix.py`

I've also included a Harper dictionary, which I've used the reduce the false
positives when using `harper-ls`.

No `requirements.txt` is present, as nothing not present in Python core is used.
